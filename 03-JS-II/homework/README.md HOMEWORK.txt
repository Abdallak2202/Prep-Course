*for: Es una expresion de bucle que posee JavaScript, se usa declarando una variable que servira como contador, una condicion 
para que se realice el bucle (por lo general el numero que posee la variable de contador), y al final se le suma 1 a la variable 
que sirve de contador; en caso de que la condicion se cumpla (sea true), se ejecutara el codigo dentro del bucle for; pero en caso
de que no se cumpla (sea false), se dejara de ejecutar el codigo dentro del bucle y saldra de este, ejecutando el codigo fuera del 
bucle.

*&&: Es un operador logico (AND) que indica que se deben cumplir las condiciones que estan antes y despues de && para que se 
ejecute el codigo; en caso de que no se cumplan estas condiciones (sean false) el codigo no se ejecuta.

*||: Es un operador logico (OR) que indica que se debe cumplir por lo menos una de las condiciones que esten antes y despues de ||
para que se ejecute el codigo; en caso de que ninguna de las condiciones se cumplan (sean false) el codigo no se ejecuta.

*!: Es un operador logico (NOT) que invierte el valor booleano que saldra de una declaracion, es decir, una declaracion true
saldra como false, y una declaracion false saldra como true.
